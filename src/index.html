A simple website package

This is a simple website package powered by gulp ... will help you setup a dev site and develop faster.

Once setup and running, the gulp watch command will launch a new browser window, and your html, sass and js files will be watched and upon any changes the browser will auto relaod.

Current Features

  • gulp
  • sass
  • browser-sync
  • and more ...
